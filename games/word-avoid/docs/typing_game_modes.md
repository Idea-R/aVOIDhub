# Typing Defense Game - Complete Game Mode Collection

## üéÆ CORE GAME MODES

### 1. **Classic Survival**
- Words approach from all directions at increasing speed
- Player has health/shields that decrease when words reach center
- Endless gameplay with difficulty scaling
- **Scoring**: Points for each word + speed bonus + accuracy bonus

### 2. **Wave Defense**
- Distinct waves of words with breaks between them
- Each wave has a theme (animals, tech terms, countries)
- Boss waves with extremely long/difficult words
- **Progression**: 10 waves per level, unlock new themes

### 3. **Time Attack**
- Fixed time limit (30s, 60s, 2min modes)
- Maximize score before time runs out
- Words don't cause damage, just lost points
- **Variants**: Blitz (15s), Marathon (5min), Endurance (10min)

### 4. **Perfect Run**
- One life only - any mistake ends the game
- Slower pace but requires 100% accuracy
- Bonus points for consecutive perfect words
- **Challenge**: How many words can you type perfectly?

## üéØ SKILL-SPECIFIC MODES

### 5. **Finger Training Academy**
**Pinky Boot Camp**
- Only words that heavily use pinky fingers (P, Q, Z, ;, etc.)
- Progressive difficulty: `zip` ‚Üí `pizza` ‚Üí `perplex` ‚Üí `pizzazz`

**Ring Finger Rehab**  
- Focus on weakest finger strengthening
- Words with W, S, L emphasis
- `slow` ‚Üí `swallow` ‚Üí `willingness`

**Index Finger Olympics**
- Words requiring precise index finger control
- Cross-hand reaches and rapid sequences
- `try`, `hug`, `rhythm`, `nymph`

**Thumb Coordination**
- Space bar timing challenges
- Words with unusual spacing patterns
- Rapid space-word-space sequences

### 6. **Pattern Mastery**
**Double Letter Destroyer**
- Only words with repeated letters
- `buzz`, `fizz`, `pizza`, `jazz`, `razzmatazz`
- Scoring bonus for consecutive doubles

**Alternating Hands**
- Words that force perfect left-right-left typing
- `typewriter`, `perpetuity`, `proprietor`
- Visual feedback showing hand alternation

**Same Hand Hell**
- Words typed entirely with one hand
- Left: `stewardess`, `reverses`, `wastewaters`  
- Right: `lollipop`, `monopoly`, `polyphony`

**Row Jumping**
- Words that require maximum vertical movement
- `zap`, `quiz`, `lynx`, `gym`
